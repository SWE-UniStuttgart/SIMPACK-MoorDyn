# SIMPACK-MoorDyn
## Introduction
This code allows coupling MoorDyn to SIMPACK in order to model Floating Offshore Wind Turbine. FOWTs 15 MW reference models of WindCrete and Activefloat platforms are used as examples.


## Dependencies
- Matlab 2018b
- Simpack 2019.2
- MoorDyn v1.01.00C
